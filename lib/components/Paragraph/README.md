# Paragraph
Paragraph is a text component for paragraph text.

## Component

| Prop Name  | Required?  | Type       | Description                         | Default |
| ---------- | ---------- | ---------- | ----------------------------------- | ------- |
| children   | No         | React.Node | Children to render in text.         | `null`  |

## Example
```javascript
<Paragraph>I am a paragraph.</Paragraph>
```
